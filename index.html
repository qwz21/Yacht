📝 업데이트 기록 (패치노트)

v3.0 산점도 직관화 & 수학 복기 강화
  • 산점도에 사분면 영역 음영(A·B·C·D) + 의미 라벨 추가 — '위치=의미'로 직관화
  • 황금 구역(B영역)이 비어 있음을 점선·강조 표시 (확률·점수 동시 높은 조합은 없다는 핵심 발견 시각화)
  • 1의 합~6의 합을 각각 독립된 점으로 세로 사다리 형태 분리 표시
  • 모든 달성 확률을 몬테카를로 시뮬레이션(20만 회)으로 재계산해 통일 (예: 포 오브 어 카인드 15%→28% 보정)
  • 복기 뷰어에 수학적 판단 3종 추가: ①기댓값 비교 ②그 주사위로 가능했던 최선의 선택 ③타이밍 전략 조언
v2.0 대규모 업데이트
  • 산점도 내 데이터 겹침(떡짐) 방지를 위한 미세 퍼짐(Jitter) 효과 도입
  • 산점도와 연동되는 '라운드 기록 뷰어' 추가 (좌우 화살표로 복기 가능)
  • 기록 뷰어 조작 시 해당 라운드의 산점도 타겟 하이라이트 기능 연동
v1.9 이전 버전
  • 산점도 겹침 현상 해결을 위한 '내 기록만 보기' 필터 스위치 추가
  • 조합 예시 툴팁을 숫자가 아닌 직관적인 주사위 기호(⚀~⚅)로 시각화 변경
v1.5 이전 버전
  • 확률 보기(수업 모드) ON 시 기록된 점수판의 성과별 스마트 음영 기능 추가 (0점: 빨강, 미달: 주황, 달성/초과: 진한 초록)
  • 점수판 조합명 옆에 직관적인 [목표 점수 뱃지] 표시
v1.4 이전 버전
  • 산점도 그래프 시인성 대폭 개선 (캔버스 세로 높이 확장)
  • 목표 기준점을 밋밋한 점 대신 직관적인 아이콘(주사위, 배 등)으로 변경
  • 그래프에 마우스를 올리면 상세 정보가 뜨는 스마트 호버 툴팁 기능 추가
v1.3 이전 버전
  • 요트(Yacht) 조합 달성 시 전용 팡파르 및 대형 사이렌 사운드 추가
  • 풀하우스 공식 룰 적용 안내 (5개 동일 숫자도 풀하우스/포오브어카인드로 인정되는 정상 룰 적용)
    • v1.1 이전 버전
    • 수학 교과서(11-3) 연계 '달성 확률 vs 획득 점수 산점도' 실시간 렌더링
    • 상단(1~6) 63점 달성 보너스(+35점) 게이지 및 성공/절망 사운드 추가
    • 조합 선택 결과(성공/애매/망함) 및 최종 게임 점수별 엔딩 브금(BGM) 추가
    • 모바일 환경 UI 깨짐 뷰 최적화 (반응형 주사위, 표 가로 스크롤)
    • AI가 유저 고정을 강제로 푸는 확률 계산 버그 완벽 수정
    v1.0 출시
    • 오프라인용(Single HTML) 요트 다이스 MVP 룰 엔진 구축
    • 몬테카를로 시뮬레이션 기반 실시간 조합 달성 확률 / 기댓값(EV) 오버레이 적용
